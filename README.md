# 🔐 Form Validation Practice – JavaScript Project

## 📋 Overview

This project is a hands-on exercise focused on **form validation** using ** JavaScript**. The goal is to strengthen understanding of JavaScript DOM manipulation, event handling, and real-time form validation techniques.

---

## 🎯 Purpose

The purpose of this project is to **practice JavaScript fundamentals** by building a user form that includes a variety of real-time validations for common input types.

---

## ✨ Features

- ✅ **Password Validation**
  - Minimum length check
  - Alphanumeric check
  - Uppercase/lowercase requirement
  - Real-time feedback

- 🔁 **Confirm Password Match**
  - Ensures password and confirm password fields match
  - Re-validates when either input is changed

- **Age Validation**
  - Checks if the input is not empty
  - Validates proper numeric input
  - Optionally enforces min/max range

-  **Email Validation**
  - Supports single and multiple comma-separated emails
  - Validates each email format individually
  - Displays invalid emails

- **Contact Number Validation**
  - Ensures valid number format (customizable)
  - Rejects non-numeric characters and improper lengths

-  **Dynamic Submit Button State**
  - Automatically disables submit button if:
    - Any required field is empty
    - Any non-empty field is invalid

## 🛠️ Tech Stack
